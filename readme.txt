Estrutura:

O site consiste em:

- Página inicial - Descrição do site.
- Fontes - Contém as fontes disponívis no site. Há uma caixa de texto em que o usuário pode digitar algum texto para pré-visualização da fonte.
- Contato - Formulário para entrar em contato com o administrador do site.
- Pedidos de fontes - Formulário para um usuário pedir que alguma fonte seja adicionada.

Funcionalidade:

O site consiste em oferecer ao usuário, de maneira simples, fontes para serem utilizadas em páginas da web ou para fazer download. Ao clicar em uma fonte, o usuário é redirecionado à página do Google Fonts, aonde lá, poderá copiar o código para utilização da fonte escolhida.
